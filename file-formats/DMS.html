--- title: Max IDs (DMS) ---

DMS files store only a single number, which is the maximum ID of some list. For example, etc/kasekirock_maxid has 999, which is the number of entries in etc/kaseki_defs.

All DMS files are found in etc/, with file names ending with _maxid.
