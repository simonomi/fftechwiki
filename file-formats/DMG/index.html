A DMG file is a list of text strings, each of which has an index number. Together, all DMG files make up one big list of strings. They can be found in the msg/ folder.

Usually, these text strings are lines of dialogue, but many are UI or status text ("The shop is now open", "You got 100 G"). They can contain special codes like $c2 or %s that control formatting, text replacement, or other fun effects.
