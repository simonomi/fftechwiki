A DMG file is a list of text strings, each of which has an index number. Together, all DMG files make up one big list of strings. They can be found in the msg/ folder.

Usually, the text strings are lines of dialogue, but many are UI or status text ("The shop is now open", "You got 100 G"). For simplicity, this page always refers to them as "dialogue".

Formatting codes

Dialogue can contain special codes that control formatting, text replacement, or other fun effects. These codes start with a $ or %, followed by a letter and possibly a number. They aren't rendered as part of the text when it's shown.

Colors

These color codes change the color of the text until the next color code, or the dialogue ends. Only 0–4 and 6 are used, but many others exist, possibly unintentionally. $ac is black, which makes it a duplicate of $c0, and $cd and $ce are extremely similar. $af is also technically black, but creates strange glitchy errors in the text. A–F are the same as 5–a, but any others not in this list are just black.
